## 컴파일 명령어

* c-language: gcc -o \[실행파일\] \[소스파일\] ex\) test.c 파일을 컴파일해서 test 실행파일을 만들때, $ gcc -o test test.c
* c++: g++ -o \[실행파일\] \[소스파일\] ex\) test.cpp 파일을 컴파일해서 test 실행파일을 만들때, $ g++ -o test test.cpp
